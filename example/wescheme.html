Blocks Help

Navigation

  • Previous block - Up
  • Next block - Down
  • Collapse block - Left
  • Expand block - Right
  • Collapse all - Shift-Left
  • Expand all - Shift-Right
  • First visible block - Home
  • Last visible block - End

Editing

  • Toggle Editing - Enter or Return
  • Cancel Editing - ESC or Shift-ESC
  • Edit any block - Cmd-Enter (Mac) or Ctrl-Enter (PC)
  • Insert Left - Ctrl-[
  • Insert Right - Ctrl-]
  • Start new Expression - (

Selection and Cut/Copy/Paste

  • Toggle selection - Space
  • Navigate and keep selection - Alt-Down, Alt-Up, etc.
  • Cut/Copy/Paste - Ctrl-X/Ctrl-C/Ctrl-V
  • Delete selection - Del or Backspace
  • Paste before active node - Shift-Ctrl-V

Search

  • Start searching - Begin typing. Focus will jump to first matching node.
  • Find next - Enter
  • Find previous - Shift-Enter
